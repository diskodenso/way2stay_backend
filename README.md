## API Endpoints

### Users

#### Create (post)
```./api/users```

<br>

#### Read (get)
##### All Users
```./api/users```
##### Single User (by username)
```./api/users/:username```

<br>

#### Update (put)
##### Single User (by username)
```./api/users/:username```

<br>

#### Delete (delete)
###### command to database wont be a delete more an update of the isActive flag
##### Single User (by username) 
```./api/users/:username```

---
<br>

### Flats

#### Create (post)
```./api/flats```

<br>

#### Read (get)
##### All Flats
```./api/flats```

```./api/flats?queryString=[filteredResult]```

##### Single Flat (by id)
```./api/flats/:id```

<br>

#### Update (put)
##### Single Flat (by id)
```./api/flats/:id```

<br>

#### Delete (delete)
###### command to database won't be a delete more an update of the isActive flag
##### Single Flat (by id) 
```./api/users/:id```

---
<br>
